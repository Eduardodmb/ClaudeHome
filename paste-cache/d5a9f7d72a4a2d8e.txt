# CMMC Assessment Web Application - Claude Code Prompt

## Project Overview
Build a comprehensive web application for managing CMMC (Cybersecurity Maturity Model Certification) assessments and compliance frameworks for CMMCUnlock.com. The application should support multiple compliance frameworks, evidence collection, risk management, third-party vendor assessments, and automated reporting.

**Domain**: CMMCUnlock.com
**Hosting**: Shared hosting environment (PHP-based)
**Framework**: Laravel (PHP) - Required due to shared hosting constraints

## Tech Stack Requirements (Shared Hosting Compatible)

### Backend Framework
- **PHP Version**: 8.1+ (check shared hosting compatibility)
- **Framework**: Laravel 10.x or 11.x
- **Database**: MySQL 8.0+ (most shared hosting providers support this)
- **Authentication**: Laravel Sanctum for API authentication with role-based access control
- **File Storage**: Laravel's filesystem with local driver (on shared hosting storage)
- **Excel Processing**: PhpSpreadsheet (Laravel Excel package)
- **PDF Generation**: DomPDF or TCPDF (Laravel-compatible)
- **Queue System**: Database driver (since Redis/Beanstalkd may not be available on shared hosting)
- **Cache**: File-based cache driver (Redis may not be available)
- **Session**: Database driver for sessions (more reliable on shared hosting)

### Frontend
- **Option 1 (Recommended for shared hosting)**: Laravel Blade templates with Alpine.js and Tailwind CSS
- **Option 2**: Vue.js 3 with Inertia.js (still Laravel-based, better UX)
- **CSS Framework**: Tailwind CSS
- **JavaScript**: Alpine.js for interactivity (lightweight, no build step needed)
- **Charts**: Chart.js or ApexCharts for dashboards

### Development Tools
- **Dependency Management**: Composer (PHP), NPM (frontend assets)
- **Asset Compilation**: Laravel Mix or Vite (compile before deployment)
- **Version Control**: Git

## SHARED HOSTING SECURITY REQUIREMENTS (CRITICAL)

### File Structure Security
```
/public_html (or /public) - Only this folder accessible via web
├── index.php (Laravel entry point)
├── .htaccess (Apache rules)
├── assets/ (compiled CSS, JS)
└── uploads/ (public files only - use .htaccess protection)

/home/cmmcunlock/ (or one level above public_html - NOT web accessible)
├── app/ (Laravel application code)
├── bootstrap/
├── config/
├── database/
├── resources/
├── routes/
├── storage/ (logs, cache, sessions, private uploads)
├── vendor/
├── .env (CRITICAL - must be outside public directory)
└── artisan
```

### Essential .htaccess Rules
Place in public_html/.htaccess:
```apache
# Disable directory listing
Options -Indexes

# Prevent access to sensitive files
<FilesMatch "^\.">
    Order allow,deny
    Deny from all
</FilesMatch>

# Protect .env file (if accidentally in public)
<Files .env>
    Order allow,deny
    Deny from all
</Files>

# Prevent access to Laravel directories
RedirectMatch 403 ^/?(vendor|storage|bootstrap|config|database|resources|routes|tests)/

# Laravel rewrite rules
<IfModule mod_rewrite.c>
    RewriteEngine On
    RewriteRule ^(.*)$ index.php [L]
</IfModule>

# XSS Protection
Header set X-XSS-Protection "1; mode=block"
Header set X-Content-Type-Options "nosniff"
Header set X-Frame-Options "SAMEORIGIN"
Header set Referrer-Policy "strict-origin-when-cross-origin"

# MIME Type sniffing protection
Header set X-Content-Type-Options "nosniff"
```

### Secure File Uploads
Place in public_html/uploads/.htaccess:
```apache
# Prevent PHP execution in upload directories
php_flag engine off

# Block dangerous file types
<FilesMatch "\.(php|php3|php4|php5|phtml|pl|py|jsp|asp|htm|shtml|sh|cgi)$">
    Order allow,deny
    Deny from all
</FilesMatch>

# Only allow specific file types
<FilesMatch "\.(jpg|jpeg|png|gif|pdf|doc|docx|xls|xlsx|ppt|pptx|txt|csv)$">
    Order deny,allow
    Allow from all
</FilesMatch>
```

### Database Security
- Use separate database user with minimal permissions (SELECT, INSERT, UPDATE, DELETE only)
- Never use root database user
- Use prepared statements (Laravel Eloquent does this automatically)
- Enable MySQL slow query log to detect performance issues
- Regularly backup database (set up automated backups with hosting provider)

### Shared Hosting Limitations & Workarounds

**Limitation 1: No SSH Access**
- Deploy via FTP/SFTP with FileZilla or similar
- Run `composer install --no-dev` locally, upload vendor folder
- Compile assets locally with `npm run build`, upload compiled files

**Limitation 2: No Cron Access (or limited)**
- Use Laravel Task Scheduler
- Create single cron job (if allowed): `* * * * * cd /home/cmmcunlock && php artisan schedule:run >> /dev/null 2>&1`
- Alternative: Use external cron service (cron-job.org) to hit https://cmmcunlock.com/cron/run endpoint

**Limitation 3: No Redis/Memcached**
- Use file-based cache: `CACHE_DRIVER=file`
- Use database for sessions: `SESSION_DRIVER=database`
- Use database for queues: `QUEUE_CONNECTION=database`

**Limitation 4: Limited PHP Extensions**
- Verify required extensions with hosting provider:
  - PDO (MySQL)
  - OpenSSL
  - Mbstring
  - Tokenizer
  - XML
  - Ctype
  - JSON
  - BCMath (for calculations)
  - GD or Imagick (for image processing)
  - ZipArchive (for Excel files)

**Limitation 5: No Command Line Access**
- Create web-based setup script (remove after installation):
  ```php
  // public_html/install.php (DELETE AFTER USE)
  <?php
  // Run migrations: /install.php?action=migrate&key=YOUR_SECRET_KEY
  if ($_GET['key'] !== 'YOUR_STRONG_SECRET_KEY_12345') {
      die('Unauthorized');
  }
  
  require __DIR__.'/../vendor/autoload.php';
  $app = require_once __DIR__.'/../bootstrap/app.php';
  
  if ($_GET['action'] === 'migrate') {
      Artisan::call('migrate --force');
      echo "Migrations completed";
  }
  ?>
  ```

### Rate Limiting (Prevent Brute Force)
In `app/Http/Kernel.php`:
```php
protected $middlewareGroups = [
    'web' => [
        // ... other middleware
        \Illuminate\Routing\Middleware\ThrottleRequests::class.':60,1', // 60 requests per minute
    ],
    'api' => [
        \Illuminate\Routing\Middleware\ThrottleRequests::class.':api',
        \Illuminate\Routing\Middleware\SubstituteBindings::class,
    ],
];

protected $middlewareAliases = [
    'throttle.login' => \Illuminate\Routing\Middleware\ThrottleRequests::class.':5,1', // 5 login attempts per minute
];
```

**Risk Calculation Matrix**:
```
Likelihood Values:
- Rare: 1
- Unlikely: 2
- Possible: 3
- Likely: 4
- Almost Certain: 5

Impact Values:
- Negligible: 1
- Minor: 2
- Moderate: 3
- Significant: 4
- Severe: 5

Risk Score = Likelihood × Impact
- 1-4: Low
- 5-9: Medium
- 10-14: High
- 15-25: Critical
```

```php
// Use Laravel Sanctum for API authentication
// config/sanctum.php
'stateful' => explode(',', env('SANCTUM_STATEFUL_DOMAINS', 'cmmcunlock.com,www.cmmcunlock.com')),
'expiration' => 60 * 24, // 24 hours

// Custom middleware: app/Http/Middleware/CheckRole.php
public function handle($request, Closure $next, ...$roles)
{
    if (!$request->user() || !in_array($request->user()->role->name, $roles)) {
        abort(403, 'Unauthorized action.');
    }
    return $next($request);
}
```
